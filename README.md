# Sample-App-Using-CoreData
📱 SwiftUI E-Commerce Demo App

A simple and modern iOS application built using SwiftUI that demonstrates authentication, local storage, custom UI components, and file handling without relying on external APIs.

🚀 Features

🔐 Authentication
Sign Up & Sign In functionality
User data stored locally using Core Data
📅 Custom Date Picker
Fully customizable date selection UI
Validation with error handling
🖼 Image & File Picker
Select images from gallery
File picker for documents (PDF, etc.)
File preview using QuickLook
File size validation (max 5MB)
💾 Local Storage
Core Data for user data
FileManager for caching images & files
🎨 UI & UX
Smooth animations
Custom reusable components
Tab-based navigation
Dynamic lists

🛠 Tech Stack
SwiftUI
UIKit (Bridging where required)
Core Data
PhotosUI / UIDocumentPicker
QuickLook
MVVM Architecture
